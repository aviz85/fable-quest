## 3. Puzzle script — onCommand logic

Score keys: `xai_junkyard_handle` = 10. **Sum = 10.** ✔

```
onCommand(verb, noun):

// --- הפאזל היחיד: קח את הידית ---
if verb=='קח' && (noun=='ידית דלת' || noun=='ידית') && !hasItem('yadit_delet') && !getFlag('xjunk_handle_taken'):
    setFlag('xjunk_handle_taken')
    addItem('yadit_delet', 'ידית דלת של הטנדר',
            'כרום מוברש, עיצוב מינימליסטי, נפלה מהטנדר אחרי 40 ק"מ. בדיוק לפי המפרט.')
    score(10, 'xai_junkyard_handle')
    say('אתה מרים את הידית. היא קלה, קרה, ומושלמת — כל מה שהטנדר שהיא נפלה ממנו לא היה.')
    say('ידית של דלת שלא נפתחת. שמור אותה. סמוך עליי. אני קריין, אני יודע דברים.')   // רמז לטוויסט
    return true

// --- קח ידית כשכבר יש ---
if verb=='קח' && (noun=='ידית דלת' || noun=='ידית') && hasItem('yadit_delet'):
    say('הידית כבר אצלך. אחת זה בדיוק המלאי העולמי — בייצור נפלו רק 40,000 כאלה, כולן מטנדרים נוסעים.')
    return true

// --- מוות: להיכנס לרקטה הנעוצה ---
if (verb=='היכנס' || verb=='פתח') && (noun=='רקטה נעוצה' || noun=='פתח הרקטה' || noun=='רקטה'):
    die(DEATH1)   // סעיף 5
    return true

// --- near-death: לדחוף את הצמיג ---
if verb=='דחוף' && noun=='צמיג ענק':
    say('דחפת את הצמיג. הוא התנדנד, שקל את האפשרויות, ובחר להתגלגל — לכיוון שלך.\nזינקת הצידה בזריזות שלא הפגנת מאז מבצע ליטני. הצמיג עבר, ריסק חבית ריקה בדיוק איפה שעמדת, ונעצר.\nהלוח מאחוריך הבהב באשמה: DAYS WITHOUT EXPLOSION: 0. טכנית זה לא היה פיצוץ. טכנית.')
    return true

// --- שימוש בידית פה (מוקדם מדי) ---
if verb=='השתמש' && (noun=='ידית דלת' || noun=='ידית') && hasItem('yadit_delet'):
    say('להשתמש בידית על מה, בדיוק? אין פה אף דלת. יש פה רק דברים שהיו פעם מחוברים לדברים אחרים.\nסבלנות. לכל ידית מגיע הרגע הקאנוני שלה.')
    return true

// --- בונוס אווירה: כיפה / שמיים ---
if verb=='הסתכל' && (noun=='כיפה' || noun=='שמיים' || noun=='מאדים'):
    say('כיפת חלל שקופה עם טלאי דבק בפינה. מתחתיה — "מאדים": אדמה אדומה מיובאת, כבידה תל-אביבית, וארנונה מלאה.\nזה לא כוכב אחר. זה נס ליציונה עם פילטר.')
    return true

return false
```

Notes for the implementer:
- `xjunk_handle_taken` הוא staging פנימי; רק `visited_xai_junkyard` הוא חוק מהמניפסט.
- אחרי הלקיחה, ה-hotspot של הידית נשאר אבל `look` שלה יכול לעבור ל: "רק שקע קטן באדמה נשאר. אפילו האבק פה מתגעגע אליה." (בדיקת `hasItem('yadit_delet')` בתוך look דינמי אם המנוע תומך; אחרת להשאיר סטטי — לא קריטי).
- `היכנס` עשוי להגיע כ-raw — לתפוס גם דרך `extra.raw.includes('היכנס')` עם "רקטה".
