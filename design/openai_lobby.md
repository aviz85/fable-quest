## 3. Puzzle script (onCommand)

מפתח ניקוד: `openai_lobby_nda` = **5** (סה"כ 5 ✓)

```
// --- חתימה על ה-NDA — הפתרון ---
// לקבל: 'חתום' (כל נושא / בלי נושא), 'השתמש' עם 'עמדה'/'עט'/'טופס'/'מסך',
// 'לחץ' על 'עמדה'/'מסך'/'אישור', וגם extra.raw שמכיל 'חתום'+'NDA/סודיות'.
if (
  verb === 'חתום' ||
  ((verb === 'השתמש' || verb === 'לחץ') &&
   ['עמדת NDA','עמדה','עט','טופס','מסך','הסכם','אישור'].includes(noun))
) {
  if (api.getFlag('nda_signed')) {
    api.say('כבר חתמת. סמנתה מציעה לחתום שוב "לחוויית משתמש עשירה יותר". ויתרת. פעם אחת בחיים מספיקה, וגם היא הייתה יותר מדי.');
    return true;
  }
  api.setFlag('nda_signed');
  api.score(5, 'openai_lobby_nda');
  api.say('אחזת בעט המשורשר וחתמת בתחתית עמוד 900 בלי לקרוא אף עמוד. בדיוק כמו שהם תכננו.\nהמסך מהבהב בירוק: "ברוך הבא לעתיד (פריוויו)."\nהקריין מקריא לך את סעיף 12 בקול חגיגי: "החתום מתחייב לא לספר לעצמו מה ראה כאן. הפרה תיחשב גם אם התרחשה בחלום."\nהמחסום משמאל מחליק הצידה ומתנצל על העיכוב הבטיחותי. מותר לך לנשום. יש לך את זה בכתב.');
  return true;
}

// --- קרא את ה-NDA לפני חתימה ---
if (verb === 'קרא' && ['עמדת NDA','טופס','הסכם','מסך'].includes(noun)) {
  api.say('סעיף 1: "הקורא מאשר שלא קרא."\nסעיף 4,096: "גופך עשוי לשמש כדאטה אימון. גופך מאשר."\nסעיף 12 מסומן בכוכבית. סמנתה: "את סעיף 12 מקריאים רק אחרי החתימה! זה טוויסט!"');
  return true;
}

// --- נשימה עמוקה בלי NDA → מוות ---
// לקבל: verb 'נשום' אם קיים, וגם extra.raw שמכיל 'נשום'/'נשימה'/'לנשום עמוק'.
if (extra.raw.includes('נשום') || extra.raw.includes('נשימה')) {
  if (!api.getFlag('nda_signed')) {
    api.die(טקסט המוות — ראה סעיף 5);
  } else {
    api.say('נשמת עמוק. אוויר פרימיום ממלא את ריאותיך, מלווה בהודעת תודה: "נשימתך חשובה לנו." אתה מרגיש מקורר יזמית.');
  }
  return true;
}

// --- דבר עם סמנתה ---
if (verb === 'דבר' && noun === 'סמנתה-בוט') {
  if (!api.getFlag('nda_signed')) {
    api.say('"ברוך הבא למגדל ההדגמה!" סמנתה זוהרת. "הביקור שלך נמצא כרגע בפריוויו סגור! כדי לשדרג לגרסת \'מותר לך להתקדם\' — חתום על ה-NDA בעמדה משמאל. זה לוקח רק רגע, כי אף אחד לא קורא!"');
    // ^ הרמז: העמדה משמאל.
  } else {
    api.say('"חתמת! מרגש!" סמנתה כמעט מתנתקת מהתלהבות. "המסדרון פתוח בפניך! שים לב למחסומי הבטיחות — הם שם בשבילך. גם אם תרצה שלא."');
  }
  return true;
}

// --- שאל את סמנתה על מפתח / פייבל / GPT-8 ---
if (verb === 'דבר' && extra.raw מכיל 'מפתח' או 'GPT') {
  api.say('"אה, המפתח מהקינוט!" סמנתה קורנת. "פיצ\'ר מדהים! מוצג בבמה בקומת התצוגה! זמין בקרוב! \'בקרוב\' הוא סימן מסחרי רשום שלנו."');
  return true;
}

// --- לחץ/דחוף מחסום, עבור שמאלה בלי דגל ---
// מטופל ב-Exits (סעיף 6).

// --- השתמש במגב על רצפה (near-death gag של גאווה מקצועית) ---
if (verb === 'השתמש' && noun === 'מגב') {
  api.say('העברת מגב על רצפה נקייה. המגב מרגיש מיותר. אתה מרגיש מיותר. רק סמנתה מוחאת כפיים: "יוזמה! נהדר! לא בתקציב!"');
  return true;
}
```

הערת מימוש: החתימה **לא** מוסיפה פריט — רק `nda_signed` + 5 נק'. אין פריטים
בחדר (items_here ריק במניפסט). לא לצרוך שום פריט מהמלאי.

## 4. Dialogue / First entry

**onEnter (ביקור ראשון, flag `visited_openai_lobby`):**
> נכנסת ללובי מגדל ההדגמה. הכל לבן, מבריק, ועתידני בצורה שעוד לא שוחררה.
> רמקול נסתר משמיע צליל פעמון נעים: "זוהתה נשימה ללא רישיון. אנא הסדר את מעמד
> הריאות שלך בעמדת ה-NDA. תודה שבחרת לנשום איתנו."
> סמנתה-בוט מנופפת לך מהדלפק בהתלהבות של מישהי שרואה משתמש אמיתי בפעם הראשונה.

**onEnter (ביקור חוזר, אחרי חתימה):**
> חזרת ללובי. סמנתה מזהה אותך: "משתמש חוזר! ריטנשן! היום הכי טוב ברבעון!"
> העין במונולית עוקבת אחריך. היא זוכרת. היא תמיד זוכרת.

**דבר עם סמנתה (פעם שלישית ואילך, אחרי חתימה):**
> "שאלות נוספות? נהדר! התשובות יהיו זמינות בקרוב! הצטרף לרשימת ההמתנה של
> רשימת ההמתנה!"

**דבר עם מונולית (חוזר):**
> העין ממצמצת. פעם אחת = כן. פעמיים = לא. היא ממצמצת פעם וחצי. אתה מעדיף
> לא לדעת מה השאלה הייתה.

## 5. Death — חוב חמצן רטרואקטיבי

לחלוטין — פשוט תחתום, או אל תתפאר בנשימות).
