# openai_boardroom — חדר הישיבות של הדלת המסתובבת (act: openai, music: openai, 20 נק')

קומה גבוהה במגדל ההדגמה. המנכ"ל מפוטר ומוחזר כל עשרים שניות בטקס דלת מסתובבת.
נורת סטטוס מעל הדלת: **ירוק = מועסק, אדום = מפוטר**. הפאזל: לתזמן בקשת שיבוץ
כ"צוות ניקיון במה" לשניות הירוקות, עם `tag_tzevet` במלאי → `setFlag('bama_access')`.
אין פריטים בחדר. יציאה יחידה: **down → openai_expo**.

**מחזור הנורה (חוק, מסונכרן בין draw ל-onCommand):** `const lit = frame % 200; ירוק אם lit < 60, אדום אחרת` (6 שניות ירוק, 14 אדום). קובץ הסצנה שומר את ה-frame האחרון ממשתנה מודול שמתעדכן ב-draw (`let curFrame; export function draw(...){ curFrame = frame; ... }`) — וה-onCommand קורא ממנו: `isGreen = (curFrame % 200) < 60`.

## 3. Puzzle script — onCommand

סה"כ ניקוד חדר: **20** = 20. מפתח יחיד: `openai_boardroom_shibutz`.

```
// עזר: isGreen = (curFrame % 200) < 60   (curFrame נשמר מ-draw, ראה §1)

// --- דבר עם המנכ"ל (הפאזל המרכזי) ---
(verb 'דבר' && noun 'מנכ"ל'):
  if getFlag('bama_access') →
    say("'אתה כבר משובץ, גיבור. צוות ניקיון במה. אל תאחר — הקינוט מתחיל כשהוא מתחיל,\nכלומר כשיהיה מוצר, כלומר בקרוב, כלומר לך כבר.'"); return true

  if !isGreen:   // הנורה אדומה — הוא מפוטר
    say("אתה מדבר אל הדלת המסתובבת. מתוכה בוקע קול נלהב:\n'מאושר! הכול מאושר! שיבוצים, תקציבים, קח גם את הרכב שלי!'")
    say("הקריין מזכיר לך בעדינות: אישור של מנכ\"ל מפוטר שווה בדיוק כמו NFT של קבלה.\nחכה לנורה הירוקה. ותפסיק לדבר עם דלתות, זה מתחיל להדאיג.")
    return true

  // הנורה ירוקה — הוא מועסק
  if hasItem('tag_tzevet'):
    setFlag('bama_access')
    score(20, 'openai_boardroom_shibutz')
    say("אתה מנצל את שניות הכהונה: 'שיבוץ. צוות ניקיון במה. יש תג.'\nהמנכ\"ל סורק את התג בקצב של אדם שיודע שהזמן שלו ספור — כי הוא ספור:")
    say("'מאושר! צוות ניקיון במה! אתה בדיוק הסינרג'יה שחיפשנ—'\nהנורה מתחלפת לאדום. הדלת בולעת אותו. חמשת חברי הבורד מוחאים כפיים.\nלא משנה — האישור נחתם בירוק. יש לך גישה לבמה. ('bama_access' הודלק. גם אתה, קצת.)")
    return true
  else:
    say("'שיבוץ? מקסים! תג צוות יש?' אתה מגשש בכיסים. מגב יש. תג אין.\n'אז זה ל-HR, קומה 12.' הוא מחייך. 'אה, לא. את HR פיטרנו. גם אותי, עוד—'\nהנורה מתחלפת לאדום באמצע המשפט. הדלת בולעת אותו. חפש מישהו עם תגים למטה, בתצוגה.")
    return true

// --- תן / השתמש תג על המנכ"ל (מסלול חלופי, אותו פאזל) ---
(verb 'תן'|'השתמש' && (noun 'תג' || noun 'מנכ"ל') && hasItem('tag_tzevet')):
  if getFlag('bama_access') → say("הוא כבר סרק את התג. פעמיים זה חשוד. גם פעם אחת היה חשוד."); return true
  if !isGreen →
    say("הושטת את התג לעבר הדלת המסתובבת. הדלת לא סורקת תגים. הדלת סורקת קריירות.\nחכה לירוק."); return true
  → אותו ענף הצלחה כמו 'דבר' (setFlag + score('openai_boardroom_shibutz') + say)
  // הערה: לא מסירים את tag_tzevet — המנכ"ל סורק ומחזיר. התג נדרש בבמה.

(verb 'תן' && noun 'תג' && !hasItem('tag_tzevet')):
  say("היית נותן תג בשמחה. אין לך. יש לך מגב, כבוד עצמי, וחוב פיקדון. אף אחד מהם לא נסרק.")
  return true

// --- מוות: כניסה לדלת המסתובבת ---
(verb 'לך'|'עבור'|'היכנס'|'דחוף' && noun 'דלת מסתובבת'):
  → die (§5, "סיבוב הקריירה")

// --- כמעט-מוות: השולחן ---
(verb 'דחוף'|'טפס' && noun 'שולחן'):
  say("טיפסת על שולחן הדירקטוריון. חמשת חברי הבורד מחאו כפיים — הם חשבו שזה קינוט.\nהנורה הבהבה אדום-ירוק-אדום בבלבול. לשנייה אחת כמעט מונית לסמנכ\"ל.")
  say("ירדת בזמן. עוד שנייה אחת שם למעלה והיית מקבל אופציות, ואז מפוטר, ואז ספר זיכרונות.\nהקריין רושם לעצמו: 'כמעט. היה כמעט.'")
  return true

// --- נורה: ניסיון לתזמן ידנית ---
(verb 'השתמש'|'משוך' && noun 'נורת סטטוס'):
  say("אתה שוקל לטפס ולתקוע את הנורה על ירוק. רעיון יפה. גם ועדת התגמול חשבה עליו.\nהיא יושבת עכשיו בקומה 3-, בלי חלונות. הנורה נשארת עצמאית.")
  return true

// --- קונפטי (אם יש; איסטר-אג, בלי ניקוד) ---
(verb 'השתמש' && noun 'קונפטי' && hasItem('konfeti')):
  say("שקלת לירות קונפטי באמצע ישיבת דירקטוריון. הקריין עוצר אותך: חמשת האלה\nימחאו כפיים כל כך חזק שהבניין יקבל סבב גיוס. שמור את זה לבמה. שם זה יהיה אמנות.")
  return true
```

```
    say("בקצה — דלת מסתובבת מזהב, ומעליה נורה: ירוק = יש מנכ\"ל, אדום = אין מנכ\"ל.\nהיא מתחלפת בקצב של סטארטאפ. שלט על הקיר: 'DAYS SINCE LAST CEO: 0'. תמיד אפס. אף פעם לא עודכן. תמיד נכון.")
    say("אתה צריך שיבוץ לבמה. המנכ\"ל מאשר שיבוצים — כשהוא קיים. תזמון, יענקל'ה. הכול תזמון.\nארבעים שנה אתה מחכה שהרצפה תתייבש. שש שניות של ירוק זה כלום בשבילך.")
```
